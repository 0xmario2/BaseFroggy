# 🐸 BaseFroggy
**Build • Fight • Earn on Base**

BaseFroggy is a strategy game built on the **Base Network** — a fast, low-cost, and secure Ethereum Layer 2. It combines fun kingdom-building gameplay with real rewards through play-to-earn mechanics.

---

## 🎮 About the Project
In BaseFroggy, you will build and expand your own frog kingdom, gather resources, train powerful armies, and compete against others. Every coin you earn in-game has real value and can be converted into the official BASEFROGGY token once connected to your Web3 wallet.

---

## ✨ Key Features
✅ **Auto Resource Generation** — Gold, Water, Lily Leaves, and Food increase automatically over time
✅ **Build & Upgrade System** — Unlock new structures, boost production, and expand your storage limits
✅ **Army Management** — Train different types of frog units with unique strengths
✅ **Battle & Raid System** — Attack other kingdoms to steal resources and earn rank points
✅ **Research & Technology** — Permanently improve production, attack, defense, and storage
✅ **Daily Missions & Rewards** — Complete tasks to earn extra resources
✅ **Wallet Integration** — Fully compatible with Base Mainnet and Web3 wallets
✅ **Auto Save Progress** — Your game state is saved automatically in your browser

## 🚀 How to Play
1. Download or copy the `index.html` file
2. Open it in any modern web browser (Chrome, Firefox, Edge, Brave, etc.)
3. Install a Web3 wallet such as **MetaMask** or **Coinbase Wallet**
4. Click **Connect Wallet** to link your wallet to the Base Network
5. Start building your kingdom and earning rewards!
